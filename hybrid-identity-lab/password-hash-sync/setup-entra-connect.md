# Azure AD Connect Setup for Password Hash Sync

## Prerequisites
- Azure AD Connect downloaded and ready
- Service account with appropriate permissions
- Network connectivity between on-prem and cloud

## Installation Steps
1. Run Azure AD Connect installer
2. Accept license terms
3. Choose Express Settings or Custom Installation
4. Configure sync scope and organizational units
5. Enable password hash synchronization option
6. Complete installation and initial sync

## Configuration
- Select forests to synchronize
- Configure sync rules
- Set up filtering options
- Enable optional features

## Post-Installation
- Monitor sync status
- Configure directory extensions if needed
- Set up alerts for sync failures
